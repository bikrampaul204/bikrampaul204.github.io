Project Description

This project is an AI based security system for human detection, recognition and tracking purpose. It was developed in python using openCV for computer vision. SQL has been used to store the data of the person being tracked such as the location of the person and the time for which the person was in that camera frame.

The system also consist of user interface for the boss/admin to login and view the data stored in the database regarding the employees and the employee could also login and view their own data.
The frontalface and the upper body cascades has been used to detect a human. The HogDescriptor is also used to detect a person in the frame. The LBPH Face Recognizer model has been used to recognize the person in the frame.
