Project Description

Frameworks/Programming Languages/Concepts: Python, Django, ARIMA, FbProphet

Generally the model is predicting the usage of network by the users of a company, and so the parameters on which it is dependent are number of current users of the company, current bandwidth usage of each user and current network capacity, therefore, the model depends on three regressors or parameters. The forecasting model used is FbProphet and the model is trained with past data of the target data and the data of the three regressors.

The number of users and the bandwidth usage is kept nearly constant but reducing the network capacity from 50000000 to 40000000, which should impact the network percentage usage. The model predicts a steep surge in the percentage usage.

The number of users and the bandwidth usage is kept nearly constant but increasing the network capacity from 50000000 to 60000000. The model predicts a down curve in the percentage usage.
